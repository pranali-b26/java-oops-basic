# Java OOPS Basics
This repository contains Java programs demonstrating Object Oriented Programming concepts like classes, objects, and methods.

## Files
- MyProgramPerson.java : Example of class `Person` with name and age

### Rectangle.java
- Demonstrates the use of **classes** and **methods** in Java.
- Prompts the user to enter the **height** and **width** of a rectangle.
- Calculates and displays the **area** using the method `calArea()`.
- Shows basic **Object-Oriented Programming (OOP)** concepts like:
  - Creating an object of a class
  - Accessing data members
  - Calling methods
